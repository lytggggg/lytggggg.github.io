CSS是级联样式表（Cascading Style Sheets）的缩写。主要用来美化HTML的内容

选择器是需要改变的HTML的元素。

css中有4种定位

fixed（固定定位），relative（相对定位），绝对定位（absolute），静态定位（static）。

fixed

当元素的position属性设置为fixed时，这个元素就被固定了，被固定的元素不会随着滚动条的拖动而改变位置。

语法：

语法：
